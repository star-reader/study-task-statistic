# 学习任务时间统计与计划程序

使用vue3开发的web程序，可以统计每日学习计划详情与总时间。配合学习计划与线上自习室效果更佳

#  如何安装使用

## 安装依赖
```
yarn install
```

### 本地调试环境
```
yarn serve
```

### 项目打包
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
